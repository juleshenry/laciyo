\chapter{Scope and Purpose}
This document is the authoritative grammar specification for Lacyo.
It defines the formal rules, morphological system, and phonological constraints
that govern the language.

\chapter{Design Principles}
\section{Optimization Priorities}
\begin{itemize}
  \item Minimum syllable length across all source languages.
  \item Minimum phoneme inventory to reduce global sound complexity.
  \item Zero grammatical collisions within class (verbs, nouns, etc.).
\end{itemize}
